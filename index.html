Spam Detection

System Information

Architecture Overview

This spam detection system uses a Multinomial Naive Bayes classifier trained on a dataset of labeled emails. The system processes text through a machine learning pipeline that converts raw text into numerical features.

Technical Implementation

  • Feature Extraction: CountVectorizer transforms text into token frequency vectors
  • Classification: Multinomial Naive Bayes algorithm applies Bayes' theorem with multinomial distribution
  • Probability Calculation: P(spam|text) = P(text|spam)P(spam)/P(text)
  • Backend: Flask REST API with JSON response format
  • Model Serialization: joblib for model persistence

Processing Pipeline

  1. Text input is tokenized and normalized
  2. Tokens are converted to numerical feature vectors
  3. Model calculates class probabilities
  4. Classification threshold determines spam/not-spam label
  5. Results are returned with probability score

Performance Metrics

The model achieves approximately 95% accuracy on test data with balanced precision and recall metrics. False positive rate is optimized to minimize legitimate messages being classified as spam.
